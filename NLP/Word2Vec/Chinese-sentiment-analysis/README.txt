项目描述

目的：根据现有的商品评论，判断新的评论情感：喜欢 or 不喜欢；
	扩展开来就是情感分析：输入一句话，输出情感 -> 腾讯情感分析：https://ai.qq.com/product/nlpemo.shtml
	
流程：
    利用 jieba 库对句子进行分词(相对于英文多了分词步骤)，句子拆成词,当作新的特征 words(英文是空格相对于自动分词了)
    根据neg.xls、pos.xls文件内容个数，自己组装 label，并使用train_test_split 拆分测试和验证集
    使用训练数据当作word2vec 语料库，然后把训练和验证集的句子映射成300维向量既特征是300维
    使用 SVM 进行二分类训练
